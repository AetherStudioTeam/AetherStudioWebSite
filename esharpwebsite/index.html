pub actor Pipeline<TIn, TOut>
{
    input source: Channel<TIn>;
    output sink: Channel<TOut>;

    using transform: Transformer<TIn, TOut>;

    on start async =>
    {
        await foreach (var message in source)
        {
            match transform.Process(message)
            {
                case Ok(result): await sink.Emit(result);
                case Error(problem): log.warn(problem, scope: "pipeline");
            }
        }
    }
}

Why teams bet on E#

From hypergrowth startups to planet-scale infrastructure providers, E# brings uncompromising performance with a language ergonomics layer that engineers love.

Zero overhead abstractions

Control when you need it

Traits lower to monomorphized native code, ensuring no virtual dispatch penalties in high-frequency execution paths. Inline assembly blocks keep deterministic control when interfacing with hardware.
