Package managers

Integrate E# with your preferred tooling. Each package is signed and published with SBOM metadata.

Homebrew macOS & Linux 
brew tap aetherstudio/esharp
brew install esharp-cli
APT Debian & Ubuntu 
echo "deb [signed-by=/usr/share/keyrings/esharp.gpg] \
https://packages.e-sharp.dev stable main" | sudo tee /etc/apt/sources.list.d/esharp.list
sudo apt update && sudo apt install esharp
Winget Windows 
winget install AetherStudio.Esharp

Nightly channel

Access experimental compiler optimizations, new runtime schedulers, and evolving language proposals.

Why nightly?

Preview features like region-based memory tuning, actor timelines, and zero-copy serialization APIs.

  • Feature flags per module
  • Enhanced tracing probes
  • Reference compiler rebuilds nightly

Install nightly

Nightly builds install alongside stable so you can validate workflows safely.

CLI Nightly channel 
esharp channel set nightly
esharp upgrade

Official container images

Optimized OCI images for CI/CD pipelines, orchestrators, and air-gapped environments.

Image Tag Includes
ghcr.io/aetherstudio/esharp stable CLI, compiler, runtime inspector
ghcr.io/aetherstudio/esharp nightly Latest compiler snapshot, proposal flags
ghcr.io/aetherstudio/esharp-runner slim Runtime only, ideal for production workloads
Docker CI example 
docker pull ghcr.io/aetherstudio/esharp:stable

docker run --rm \
  -v "$PWD":/workspaces/project \
  ghcr.io/aetherstudio/esharp:stable \
  esharp build --release
