Lifecycle hooks

Lifecycle events ensure deterministic startup and teardown across services. Hooks run in dependency order and expose metrics automatically.

services/ingestor.es Lifecycle hooks 
service Ingestor
{
    requires broker: MessageBroker;
    config batchSize: Int = 512;

    on configure => broker.Connect("telemetry");

    on ready async =>
    {
        await broker.Stream("telemetry", handle);
    }

    fn handle(packet: Packet) async
    {
        await pipeline.Forward(packet)
             .onError(|err| log.error(err, scope: "ingestor"));
    }
}

Runtime insights

Monitor scheduler queues, evaluate contract adherence, and map latency budgets directly from your cluster. Visualizations refresh in near real time while the inspector stays lightweight.

Inspector Purpose Typical usage
orbit top Snapshot actor throughput and mailbox depth. Pinpoint hotspots across services before scaling out.
orbit traces Stream distributed traces with contract violations flagged. Debug cascading failures with exported flamegraphs.
orbit qa Run contract suites and performance gates in isolation. Validate pre-production workloads with synthetic traffic.
